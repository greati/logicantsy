Detailed Description

Gives a judgement-value correspondence given when they are represented in a billatice, the judgements occuppying the edges of the underlying graph.

For example:

{ { 0, {{1, N}, {2, Y}}, 1, {{0, N}, {3, nY}}, 2, {{0, Y}, {3, nN}}, 3, {{1, nY}, {2, nN}} } }
