Detailed Description

template<typename CellType = int>
class ltsy::TruthTable< CellType >

Represents an NValues-valued Arity-ary truth table.

Values are taken to be 0,...,NValues-1.

Only the images are stored. Each input tuple is represented by a natural number corresponding to the position of the tuple in the lexicographical order.
