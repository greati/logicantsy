Detailed Description

template<typename CellType = int>
class ltsy::TruthInterp< CellType >

Holds the interpretation relation between a connective and a truth table.

Main purpose is to guarantee that this relation is well set up.

Notice that CellType will define if the interpretation is deterministic or not.
