\doxysubsection{Detailed Description}
Multiply a number by a factor using given mapping. 

Can be used to write transforms for S\+I\+ZE or D\+U\+R\+A\+T\+I\+ON inputs.

Example\+: With mapping = {\ttfamily \{\char`\"{}b\char`\"{}-\/$>$1, \char`\"{}kb\char`\"{}-\/$>$1024, \char`\"{}mb\char`\"{}-\/$>$1024$\ast$1024\}} one can recognize inputs like \char`\"{}100\char`\"{}, \char`\"{}12kb\char`\"{}, \char`\"{}100 M\+B\char`\"{}, that will be automatically transformed to 100, 14448, 104857600.

Output number type matches the type in the provided mapping. Therefore, if it is required to interpret real inputs like \char`\"{}0.\+42 s\char`\"{}, the mapping should be of a type $<$string, float$>$ or $<$string, double$>$.
