\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Cell\+Type = int$>$\newline
class ltsy\+::\+Truth\+Table$<$ Cell\+Type $>$}

Represents an {\ttfamily N\+Values}-\/valued {\ttfamily Arity}-\/ary truth table. 

Values are taken to be 0,...,{\ttfamily N\+Values}-\/1.

Only the images are stored. Each input tuple is represented by a natural number corresponding to the position of the tuple in the lexicographical order.
