\doxysubsection{Detailed Description}
Represents a generalized version of matrices, in the following sense\+: 


\begin{DoxyItemize}
\item we may define more than one distinguished set (thus allowing the creation of q-\/matrices, p-\/matrices, B-\/matrices...)
\item we may interpret the connectives using (partial) non-\/deterministic truth-\/tables.
\end{DoxyItemize}
